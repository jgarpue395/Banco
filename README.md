# Banco

## Descripción
Este programa realiza diferentes operaciones en una cuenta bancaria y muestra información de las mismas.

## Ayuda de uso
El programa usa diferentes metodos para realizar las distintas operaciones.
- **addAccount:** crea una cuenta.
- **removeAccount:** elimina una cuenta.
- **creditAccount:** concede un credito a la cuenta correspondinete al numero de cuenta indicado, de la cantidad puesta.
- **whitdraw:** reduce el dinero de la cuenta que tiene el número de cuenta indicado, en la cantidad puesta.
- **depositAccount:** aumenta el dinero que hay de la cuenta ue tiene el número de cuenta indicado, en la cantidad puesta.
- **transfer:** reduce el dinero indicado en la cuenta que coincide con el número de cuenta indicado y lo suma en la segunda.
- **displayAccount:** muestra por pantalla la información de una cuenta, pasandole el número de cuenta.
- **listOperationAccount:** muestra las operaciones realizadas en una determinada cuenta, después de la fecha indicada.
